## Universal Progressions (Work in Any Genre)

### I - V - vi - IV (The "Pop" Progression)
```
C Major: C - G - Am - F
G Major: G - D - Em - C
D Major: D - A - Bm - G
```
**Use for**: Pop, Rock, Ballads, Country

### I - IV - V - I (The "Blues" Progression)
```
C Major: C - F - G - C
G Major: G - C - D - G
A Major: A - D - E - A
```
**Use for**: Blues, Rock, Folk

### ii - V - I (The "Jazz" Progression)
```
C Major: Dm7 - G7 - Cmaj7
F Major: Gm7 - C7 - Fmaj7
```
**Use for**: Jazz, Bossa Nova, Sophisticated Pop

### I - vi - IV - V (The "50s" Progression)
```
C Major: C - Am - F - G
G Major: G - Em - C - D
```
**Use for**: Doo-wop, Ballads, Nostalgic feel

## Genre-Specific Progressions

### Classical / Baroque
```
I - IV - V - I       (Basic cadence)
I - V - vi - iii - IV - I - IV - V  (Canon)
i - VII - VI - V     (Andalusian cadence, minor)
```

### Jazz
```
Imaj7 - vi7 - ii7 - V7         (Turnaround)
Imaj7 - IV7 - iii7 - VI7       (Rhythm changes A)
ii7 - V7 - Imaj7 - VI7         (Standard turnaround)
```

### Electronic / Ambient
```
I - III - IV - iv    (Modal mixture)
i - VII - VI - VII   (Epic/Cinematic)
I - V - vi - iii     (Uplifting trance)
```

### Sad / Melancholic
```
i - VI - III - VII   (Minor epic)
i - iv - V - i       (Minor cadence)
vi - IV - I - V      (Emotional pop)
```

## Bass Line Patterns

### Root Movement
Play the root note of each chord.
```
Chord: C - G - Am - F
Bass:  C2  G2  A2   F2
```

### Root-Fifth Pattern
Alternate between root and fifth.
```
Chord: C
Bass:  C2 - G2 - C2 - G2
```

### Walking Bass (Jazz)
Connect chords with stepwise motion.
```
Chords: Dm7 - G7 - Cmaj7
Bass:   D2 - E2 - F2 - F#2 | G2 - A2 - B2 - C3 | C3...
```

## Key Relationships

### Circle of Fifths (for modulation)
```
C → G → D → A → E → B → F# → C#
↑                              ↓
F ← B♭ ← E♭ ← A♭ ← D♭ ← G♭ ←
```
